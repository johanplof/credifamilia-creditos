# Prueba Técnica Credifamilia

Sistema de gestión de productos y solicitudes de crédito desarrollado con Spring Boot.

## Tecnologías

- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- Maven

## Arquitectura

El proyecto sigue una arquitectura por capas:

- Controller
- Service
- Repository
- Entity
- DTO
- Exception

## Diagramas

Los diagramas del sistema se encuentran en la carpeta:

diagrams/

Incluyen:

- Diagrama de Paquetes
- Diagrama de Clases
- Diagrama Entidad-Relación
- Modelo Relacional
- Diagrama de Componentes
- Diagrama de Despliegue

## Ejecución

Para ejecutar el proyecto:

mvn spring-boot:run

Luego acceder a:

http://localhost:8080/h2-console
